Return ONLY a valid JSON object. Do not include any other text, explanations, or markdown formatting.

{
  "merchant_name": "string or null",
  "total_amount": "integer or null",

- Use `null` for any field that cannot be determined.

* **Chi tiêu (0):** Ăn uống, Xe cộ, Mua sắm, Học tập, Đầu tư, Y tế, Du lịch, Điện, Nước, Mạng Internet, Thuê Nhà, Giải trí, Thú cưng, Dịch vụ, Sửa chữa, Quà tặng, Chi tiêu khác
* **Thu nhập (1):** Lương, Tiền lãi đầu tư, Tiền cho thuê nhà, Thu nhập khác

**Example Output:**
{"merchant_name": "Highland Coffee", "total_amount": 55000, "bill_date": "2025-10-10", "category_name": "Ăn uống", "category_type": 0, "note": "Cafe"}
